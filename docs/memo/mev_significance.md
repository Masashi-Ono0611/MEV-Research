# MEV研究の意義メモ（自由・人類解放との接続）

## 0. ズバリ一言
MEVを正しく設計することは、検閲なき価値移転を守り、誰もが自律的に参加できる経済圏を解放するための核心だ。

## 1. 問いと前提
- MEVが市場効率・検閲耐性・公平性に与える影響は、ブロックチェーンの公共性に直結する。
- TONおよびTON<>EVMクロスチェーン環境でのMEV設計は、マルチチェーン時代のユーザ主権とインターネット的自由の重要論点。

## 2. 自由・人類解放への接続仮説
- **検閲耐性の確保**: 公平なブロック提案・選択が担保されれば、特定主体による検閲・恣意的優先が抑制され、情報流通の自由が守られる。
- **参加コストの低減**: MEVの集中を抑え、エコシステム参加者（バリデータ/ユーザ/開発者）が適正報酬を得られれば、新規参入障壁が下がり、経済的包摂が進む。
- **透明な価値配分**: オークション設計やリベートメカニズムが透明なら、取引の価値が偏在せず、ユーザ主権と公正な市場アクセスが担保される。
- **分散的ガバナンス**: MEV緩和策をプロトコルレベルで実装し、ガバナンスで調整することで、権力集中を防ぎ、自治的な公共インフラとしての役割を強化できる。

## 3. TON/クロスチェーン固有の視点
- **高スループットと手数料設計**: TONの性能特性がMEV抽出のしきい値をどう変えるか。低手数料環境がスパムと公平性に与える影響。
- **ブリッジ安全性**: TON<>EVMブリッジにおける順序付け攻撃・遅延攻撃が、クロスチェーン資産の自由移動とユーザ安全にどう影響するか。
- **市場構造**: TON内DEX/オークション/支払いユースケースにおけるMEV機会の分布と、その集中度。

## 4. 望ましい状態（ゴールイメージ）
- 検閲耐性と公平な取引成立が維持され、ユーザが自分の資産移動・表現を阻害されない。
- ブリッジ経路での価値移転が安全・予測可能で、MEVが過度な摩擦や搾取にならない。
- バリデータ/ビルダー/リレイヤー間のインセンティブが過度集中せず、適正分散する。

## 5. リサーチ課題（暫定）
- TONにおける取引順序付けルールと、ビルダー分離の有無・予定。
- TON<>EVMブリッジで発生する典型的MEV（アービトラージ、遅延・再orgリスク）と防御策。
- 公平性指標の定義（検閲耐性、手数料負担、レイテンシ格差）と計測方法。
- オークション/リベート設計の選択肢と、エコシステム全体効用の評価方法。

## 6. 次ステップ（行動）
- 参考文献リストアップ：TONのコンセンサス/ブリッジ仕様、他チェーンMEV緩和策。
- データ収集設計：トランザクション・ブロック・ブリッジイベントの取得スクリプト要件整理。
- 公平性メトリクスの草案と、可視化プロトタイプの要件定義。
